## Test Retriever

![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)

This repository contains code to test the retriever for the [HCMUT Chatbot](https://www.ura.hcmut.edu.vn/bk-tvts/), which utilizes the RAG (Retrieval-Augmented Generation) technique. The primary goal of this code is to identify any bugs and verify the functionality of the retriever.

> **Note:** For testing purposes, I am using my own free trial QDRANT database, which has the same structure as the main database used by the Chatbot.
