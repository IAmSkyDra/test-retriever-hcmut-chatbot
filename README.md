## Test Retriever
![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)

This repository contains code to test the retriever for the HCMUT Chatbot, which is based on the RAG (Retrieval-Augmented Generation) technique. The purpose of this code is to identify bugs and verify the features of the retriever :)))

> **Note**: I am using my own free trial database of QDRANT instead of the main database used by the Chatbot.
